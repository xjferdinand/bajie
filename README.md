# 🎯 八戒X值智能匹配系统

一个高效的房号位子推荐工具，基于你提供的X值和位子优先级，快速判断最佳进场位子。

## 💡 工作原理

### 核心逻辑

```
输入房号 (5位数字 ABCDE)
    ↓
按位置检查 vs X值列表
    ↓
筛选出所有匹配的位子
    ↓
按优先级排序
    ↓
输出最优推荐 + 最多2个次选位（共3个）
```

### X值配置

| 位子 | 需要的X值 | 检查房号的哪一位 |
|------|---------|------------------|
| 2号 | 2, 4 | 第2位 (B) |
| 3号 | 8, 7 | 第3位 (C) |
| 4号 | 4, 5, 8 | 第4位 (D) |
| 5号 | 3, 9 | 第5位 (E) |
| 6号 | 9, 6, 1 | 第5位 (E) |

### 优先级顺序

```
🥇 6号 (优先级 1) - 最优 [综合评分0.95]
🥈 5号 (优先级 2)      [综合评分0.69]
🥉 3号 (优先级 3)      [综合评分0.59]
4️⃣ 4号 (优先级 4)      [综合评分0.49]
5️⃣ 2号 (优先级 5)      [综合评分0.44]
```

## 📱 使用示例

### 例1：全匹配

```
输入：74759
拆分：7-4-7-5-9

检查结果：
✓ 2号位(B=4)：4在[2,4] → 匹配
✓ 3号位(C=7)：7在[8,7] → 匹配
✓ 4号位(D=5)：5在[4,5,8] → 匹配
✓ 5号位(E=9)：9在[3,9] → 匹配
✓ 6号位(E=9)：9在[9,6,1] → 匹配

推荐：
🟢 最优推荐：6号位 (X=9)
🔵 次选位：5号位 (X=9)、3号位 (X=7)
```

### 例2：部分匹配

```
输入：12345
拆分：1-2-3-4-5

检查结果：
✓ 2号位(B=2)：2在[2,4] → 匹配
✗ 3号位(C=3)：3不在[8,7] → 不匹配

推荐（按优先级）：

```

## 🔧 自定义配置

编辑 `index.html`，找到以下两个配置对象：

### 修改X值需求

```javascript
const X_VALUES = {
  2: [2, 4],        // 2号需要的X值
  3: [8, 7],        // 3号需要的X值
  4: [4, 5, 8],     // 4号需要的X值
  5: [3, 9],        // 5号需要的X值
  6: [9, 6, 1]      // 6号需要的X值
};
```

### 修改优先级顺序

```javascript
const SEAT_PRIORITY = {
  6: 1,  // 数字越小优先级越高
  5: 2,
  3: 3,
  4: 4,
  2: 5
};
```

## 🔍 常见问题

**Q: 怎样改X值？**
